# Images

Place real photography here. Replace all `img-placeholder` divs in HTML with `<img>` tags pointing to these files.

## Needed files
- `hero.jpg` — full-bleed hero (bride, editorial, soft light) — recommended 1920×1080px+
- `artist-headshot.jpg` — editorial portrait, 3:4 ratio — recommended 800×1067px
- `og-image.jpg` — Open Graph share image — 1200×630px
- `instagram-1.jpg` through `instagram-6.jpg` — square crop Instagram previews
